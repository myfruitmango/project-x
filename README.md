# Project X 🚀

## ✨ Starting

using npm:

```bash
npm install
```

or if you using yarn:

```bash
yarn install
```

> Create new env

## 🐢 Run

### Local

> for development in local

```bash
npm start
```

or

```bash
pm2 start app.local.config.js
```

### Server Staging

> for staging

```bash
pm2 start app.dev.config.js
```

### Server Production

> for production

```bash production
pm2 start app.prod.config.js
```

## 🍔 Build

Deploy

```bash
npm run build
```
